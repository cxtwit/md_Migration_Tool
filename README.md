# Markdown小助手 (Markdown Assistant)

> 一站式 Markdown 图片与链接管理工具，让文档整理更优雅。

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![GUI](https://img.shields.io/badge/GUI-PySide6-green)
![License](https://img.shields.io/badge/License-MIT-orange)

**Markdown小助手** 是一款专为创作者打造的本地化工具。它可以帮你自动化处理文档中的图片引用、清理冗余资源、批量重命名以及跨目录迁移文档，彻底解决“图片乱放”、“链接失效”和“文件名混乱”的痛点。

---

## ✨ 核心功能

### 1. 🔍 资源审计 (Asset Audit)
* **智能扫描**：分析 Markdown 文件引用的图片，精准找出文件夹中未被使用的“僵尸图片”。
* **安全清理**：提供“移动到备份文件夹”和“永久删除”两种模式，防止误删。
* **数据统计**：实时显示 MD 文件数、有效引用数和冗余文件数。

### 2. ⇄ 迁移合并 (Migration)
* **文档搬家**：将散落在各处的 MD 文件迁移到新目录，自动修正图片路径。
* **远程转本地**：自动下载文档中的网络图片（http/https）到本地，防止图床挂掉。
* **合并输出**：支持将多个 MD 文件合并为一个长文档。
* **资源隔离**：可选将图片按文档名存入独立子文件夹。

### 3. 🏷️ 批量重命名 (Batch Rename)
* **可视化规则**：支持 `{原名}`、`{序号}`、`{日期}` 等占位符组合。
* **实时预览**：在执行前即可看到重命名后的效果。
* **后悔药**：支持“撤销”上一步重命名操作。

### 4. 📦 原地整理 (Inplace Fix)
* **一键归档**：不移动文档，直接将当前目录下的散乱图片归档到 `./images` 文件夹。
* **路径修复**：自动更新文档内的图片引用链接。

## 🛠️ 安装与运行

### 1. 环境准备
本项目基于 Python 3 开发。建议使用 Python 3.10 或更高版本。

### 2. 安装依赖
在项目根目录下运行以下命令安装所需库：

```bash
python -m pip install -r requirements.txt
```
